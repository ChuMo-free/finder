🏛️ 非遗地址坐标查找工具 - 启动说明
========================================

【Windows 一键启动】
1. 双击「启动.bat」文件
2. 首次启动会自动创建虚拟环境并安装依赖（约1-2分钟）
3. 服务启动后会自动打开浏览器访问 http://localhost:5000
4. 命令行窗口会最小化运行，不影响使用

【停止服务】
- 直接关闭命令行窗口即可停止服务
- 或者在命令行窗口按 Ctrl+C

【常见问题】

Q: 双击后提示"未检测到Python"
A: 请先安装Python 3.8或更高版本
   下载地址: https://www.python.org/downloads/
   安装时请勾选"Add Python to PATH"

Q: 提示端口5000被占用
A: 可能有其他程序占用了5000端口
   可以关闭其他占用端口的程序，或者修改app.py中的端口号

Q: 浏览器没有自动打开
A: 可以手动打开浏览器，访问 http://localhost:5000

Q: 首次启动很慢
A: 首次启动需要下载安装依赖包，请耐心等待
   后续启动会快很多

Q: 虚拟环境是什么？
A: 脚本会自动创建venv虚拟环境，用于隔离项目依赖
   不会影响系统中的其他Python项目

【手动启动方式】
如果一键启动脚本有问题，可以手动启动：

1. 打开命令行（cmd或PowerShell）
2. 进入项目目录: cd 项目路径
3. 安装依赖: pip install -r requirements.txt
4. 启动应用: python app.py
5. 打开浏览器访问: http://localhost:5000

【使用步骤】
1. 上传Excel文件（包含非遗项目名称）
2. 配置字段映射和高德地图API Key
3. 开始处理，系统会自动搜索地址
4. 人工审核搜索结果和坐标结果
5. 全部完成后导出Excel

【高德API Key申请】
访问 https://lbs.amap.com/ 注册账号
创建应用，选择「Web服务」类型的Key
复制Key填入配置页面即可
