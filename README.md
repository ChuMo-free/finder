# 🏛️ 非遗地址坐标查找工具

一个基于 Flask 的 Web 应用，用于批量查找非遗项目的真实地址并转换为地理坐标。支持人工审核机制，确保数据准确性。

## ✨ 功能特性

- 📊 **Excel 批量上传**：支持 .xlsx/.xls 格式，自动识别字段
- 🔍 **智能地址搜索**：自动网络搜索非遗项目地址，提取关键信息
- 👀 **人工审核机制**：搜索结果和坐标转换结果均需人工确认
- 📍 **高德地图API**：使用高德地图进行地址地理编码
- 🗺️ **地图预览**：支持在地图上预览坐标位置
- ⏸️ **任务控制**：支持暂停、继续、中断、单条重试
- 📝 **实时监控**：进度条、日志输出，全流程可视化
- 📥 **结果导出**：一键导出包含地址和坐标的 Excel 文件

## 🚀 快速开始

### 环境要求

- Python 3.8+
- 高德地图 Web 服务 API Key（免费申请）

### 安装步骤

1. **克隆或下载项目**

```bash
cd heritage-address-finder
```

2. **安装依赖**

```bash
pip install -r requirements.txt
```

3. **申请高德地图API Key**

   - 访问 [高德开放平台](https://lbs.amap.com/)
   - 注册账号并创建应用
   - 选择「Web服务」类型的 Key
   - 复制你的 API Key

4. **启动应用**

```bash
python app.py
```

5. **访问应用**

   打开浏览器访问：`http://localhost:5000`

## 📖 使用指南

### 第一步：上传Excel文件

1. 点击「选择文件」或拖拽 Excel 文件到上传区域
2. 系统会自动读取文件并显示行数和字段列表

> 💡 Excel 文件至少需要包含一列「项目名称」

### 第二步：字段映射与API配置

1. **字段映射**：选择 Excel 中对应「项目名称」的列
2. **高德API配置**：输入你的高德地图 Web 服务 API Key
3. 点击「开始处理」

### 第三步：处理与审核

系统会自动处理每条数据，遇到需要人工审核的地方会暂停并弹出审核窗口：

#### 地址审核
- 系统会显示搜索结果列表
- 点击任意结果可自动填充地址
- 也可以手动输入或修改地址
- 确认无误后点击「确认地址」
- 如果搜索结果不理想，可以点击「重新搜索」

#### 坐标审核
- 系统会显示高德地图返回的坐标结果
- 点击任意结果可自动填充坐标
- 点击「在地图上预览」可以查看位置
- 确认无误后点击「确认坐标」
- 如果结果不理想，可以点击「重新转换」

### 任务控制

在处理过程中，你可以：

- **⏸️ 暂停**：暂停当前任务
- **▶️ 继续**：继续已暂停的任务
- **⏹️ 中断**：终止任务（已处理的数据会保留）

### 第四步：查看结果与导出

1. 所有数据处理完成后，会显示结果汇总
2. 查看每条数据的处理状态
3. 点击「查看地图」可以在地图上预览位置
4. 点击「导出Excel」下载结果文件

导出的 Excel 文件包含：
- 原始数据的所有列
- 「确认地址」列
- 「经度」列
- 「纬度」列
- 「处理状态」列

## ❓ 常见问题

### Q: 高德地图API Key 怎么申请？

A: 访问 [高德开放平台](https://lbs.amap.com/)，注册账号后在「控制台」创建应用，选择「Web服务」类型即可获取 Key。个人开发者有免费额度。

### Q: 搜索结果不准确怎么办？

A: 你可以：
1. 在审核时手动修改地址
2. 点击「重新搜索」获取新的结果
3. 直接输入你知道的正确地址

### Q: 可以处理多少条数据？

A: 理论上没有限制，但建议单次不要超过 1000 条，因为每条都需要人工审核，数据太多会比较耗时。

### Q: 任务中断后数据会丢失吗？

A: 不会。中断后已处理的数据会保留，你可以查看已完成的结果并导出。

### Q: 支持哪些 Excel 格式？

A: 支持 .xlsx 和 .xls 格式。

## 📝 注意事项

1. **网络搜索**：本工具使用公开搜索引擎进行地址搜索，搜索结果仅供参考，请务必人工审核
2. **API 配额**：请注意高德地图 API 的调用次数限制，避免超出免费额度
3. **数据隐私**：所有数据仅在本地处理，不会上传到任何第三方服务器（除了高德地图 API 调用）
4. **浏览器兼容性**：建议使用 Chrome、Firefox、Edge 等现代浏览器
